# NARIYUGG-empowerring-women
NARIYUGG is a free, women-only business support website for women who want to start or grow small, micro, rural, and low-income businesses in India.

# What works now
1. Welcome page with clear platform purpose
2. Language selection for English, Hindi, Bengali, Tamil, Telugu, and Marathi
3. Women-only self-declaration login flow
4. Business profile form based on experience, business size, location, sales, and digital comfort
5. Dashboard with recommendations by experience level
6. Free scheme/resource links
7. Simple selling price calculator

# Note about women-only login
The current website is static, so it can only use a self-declaration checkbox. A production version should connect a real backend and use a respectful verification method, such as SHG/NGO approval, invite verification, or document verification where legally appropriate.
